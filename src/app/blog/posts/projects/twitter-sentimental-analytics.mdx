## Project Overview

This project involved developing a comprehensive Twitter sentiment analysis system that streamed and analyzed over 100,000 live tweets to understand public sentiments regarding COVID-19. The system successfully categorized data into 5 unique categories and achieved an impressive prediction accuracy of over 89% on testing data.

## Key Achievements

### Data Processing
- **Streamed large datasets** with over 100k live tweets to ascertain public sentiments on COVID-19
- Implemented real-time data streaming and processing capabilities

### Classification & Accuracy
- **Categorized data into 5 unique categories** enabling nuanced sentiment analysis
- **Achieved a prediction accuracy of over 89%** on testing data

### Advanced Emotion Classification
- **Classified multiple different human emotions** when compared to the majority of binary/non-binary traditional sentimental analytic models
- Developed a multi-class classification system that goes beyond simple positive/negative sentiment

## Technical Skills

- **NumPy**
- **Natural Language Processing (NLP)**
- Real-time Data Streaming
- Machine Learning Classification
- Sentiment Analysis

## Impact

This project demonstrates advanced capabilities in sentiment analysis, moving beyond traditional binary models to provide more nuanced insights into public opinion and emotional responses during the COVID-19 pandemic. The high accuracy rate and multi-category classification make it valuable for understanding public sentiment trends.
