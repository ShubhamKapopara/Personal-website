## Project Overview

This project involved developing a Sign Language Recognition system designed to assist speech-impaired individuals. The system utilizes Convolutional Neural Networks (CNN) to recognize sign language gestures and achieved an impressive accuracy of over 90% under various challenging conditions.

## Key Achievements

### Dataset Collection
- **Gathered a diverse dataset** of sign language gestures from different individuals
- Ensured dataset diversity to improve model robustness

### CNN Implementation
- **Implemented Sign Language Recognition system** for speech impaired people utilizing CNN
- Developed deep learning model optimized for gesture recognition

### Performance Assessment
- **Assessed the system's performance** under various conditions, including different lighting, backgrounds, and hand orientations
- **Achieved an accuracy of over 90%** across diverse testing scenarios

## Impact

This Sign Language Recognition system provides a valuable tool for improving communication accessibility for speech-impaired individuals.
